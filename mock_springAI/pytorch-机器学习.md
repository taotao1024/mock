# Pytorch
PyTorch 是一个基于 Python 的开源机器学习框架


深度学习模式： 最终训练后的结果是卷积因子 也被称为权重 --> 种类单一的结果 智能识别到一种


有监督和无监督

卷积因子 --> 手动设置

损失函数 计算差距




pytorth是深度学习模型的平台    大模型计算平台 提供了标准的卷积算法、算子的实现(卷积、池化(升维和降维))


需要使用算子   搭建一个计算模型(Pytorth里的函数)   通过pytorth 和 损失函数、卷积算法 搭建计算模型。

计算模型的计算结果 和 指定结果 通过损失函数 计算得到一个差距 （约趋近于0 证明结果越正确） 通过反向传播函数 更新模型中权重的参数

计算方式、梯度下降函数、数据集(接口实现)、权重
